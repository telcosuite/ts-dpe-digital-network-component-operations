## Cross-App Flow

Digital experiences consume the same API layer used by partners. They do not own core customer, product, order, billing, inventory, or assurance records. Partner and marketplace apps manage ecosystem access, partner offers, partner orders, Open Gateway flows, usage, settlement, and support. Developer portal capabilities expose governed API products, subscriptions, sandboxing, testing, analytics, and conformance.

The original concise app overview files remain beside these folders and are linked from each app detail pack.
